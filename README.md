# Web API using JWT (Bearer Token)
## ASP.NET Core 3.1
## Instruction:
### - Open Postman
### - Get token: localhost:port//api/auth/token (POST with username: admin || password: admin)
### - Get data: localhost:port//api/values (GET)
### - Add token into Authorize header
